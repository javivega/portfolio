What is Soundsquad?

Music lovers love to go to concerts, but sometimes it is difficult to find friends to go with. Soundsquad is the platform where music lovers find each other to share new motivating experiences.

Discovery and understanding

When we received ther brief we were given a lot of functionalities, but not so much additional information. The main challenge for this project was to help to aim and focus the main goal of this app for the users in a very short time.

To face this challenge we quickly validate with the stakeholders the main idea of the app and how could be the possible structure using tools as sitemaps or user flows.
